# Váriáveis e Tipos de Dados em Python


Variáveis -> Nome de coisas (categorias,tipos, classificação) ; Coisas que vão variar

Programação é : 

-> Receber dados
-> Armazenar dados
-> Processar dados
-> Mostrar os dados processados 

Tipos de dados : 

-> Int (inteiro)
-> float (decimal)
-> Str (string) - conjunto de caractéres 
-> Bool (boleano) - True or False
